# 🤖 AI Resume Analyzer Pro

AI Resume Analyzer Pro is a Flask-based web application that analyzes PDF resumes and provides ATS scores, skill detection, job role prediction, and resume improvement suggestions.

---

## 🚀 Features

- PDF Resume Upload
- Resume Text Extraction
- ATS Score
- Skill Detection
- Job Role Prediction
- Resume Suggestions
- Download PDF Report
- Professional Dashboard

---

## 🛠 Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- pdfplumber
- ReportLab
- Chart.js
